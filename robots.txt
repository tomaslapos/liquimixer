# ============================================
# WHAT IS LIQUIMIXER
# ============================================
# LiquiMixer is a professional online calculator for mixing e-liquids
# (electronic cigarettes) and hookah/shisha tobacco. It works as a PWA
# (Progressive Web App) — runs in any browser on any device (phone,
# tablet, PC) and can be installed like a regular app without an app
# store. Works fully offline after first load. Updates automatically.
#
# Mix your own liquid or shisha blend. Safely. Precisely. At a fraction
# of the cost. The calculator tells you exactly what and how much — even
# if you've never done it before. Pick a flavor, set parameters, mix.
# The correct percentage fills in automatically.
#
# ============================================
# 9 CALCULATORS (5 eLiquid + 4 Shisha/Hookah)
# ============================================
#
# --- eLiquid (5 calculators) ---
#
# 1. LIQUID — basic e-liquid calculator for 1 flavor
#    Enter volume (how much liquid you want to mix), select nicotine type
#    (booster or salt) and its strength, pick a flavor from database of
#    5000+ concentrates — the recommended percentage fills in automatically.
#    Set your desired VG/PG ratio with a slider. Add optional additives.
#    Hit MIX → get exact amounts of every ingredient in ml and grams.
#    Supports separate PG + VG bases or premixed base.
#
# 2. LIQUID PRO — advanced multi-flavor mixing
#    Mix up to 4 flavors simultaneously, each with its own percentage.
#    Advanced flavor composition breakdown — each flavor can specify its
#    own PG/VG/alcohol/water ratio (find on manufacturer website: Capella,
#    TFA, Flavor West, Inawera, etc.). The calculator factors all of this
#    into the true final VG/PG ratio.
#    Additives: coolant (WS-23, Koolada), sweetener (sucralose, stevia),
#    flavor enhancer, terpenes.
#
# 3. SHAKE & VAPE — for longfill bottles
#    You have a pre-made flavor bottle (longfill) and need to know how
#    much base to add and how to get your target nicotine strength.
#    Enter bottle size, flavor volume, nicotine settings → result shows
#    how much VG, PG, and nicotine to pour in.
#
# 4. SHORTFILL — nicotine shot calculator
#    You have a nicotine-free bottle and need to add nicotine shots.
#    Calculator tells you exactly how many shots you need for your target
#    strength. Warns if it would overflow the bottle.
#
# 5. DILUTE — nicotine base dilution
#    Dilute concentrated nicotine base to a lower strength, e.g. 72 mg/ml
#    down to 20 mg/ml. Calculator shows how much PG and/or VG to add.
#
# --- Shisha / Hookah (4 calculators) ---
#
# 6. MIX TOBACCOS — blend commercial tobaccos
#    Select your hookah bowl size (10–25g). Search tobaccos from a
#    database of 826 entries (Al Fakher, Adalya, Darkside, Fumari,
#    Tangiers, Starbuzz, and many more). Add 2–4 tobaccos and set the
#    ratio with sliders. The calculator shows exact grams for each.
#    Save your favorite mix as a recipe you can repeat anytime.
#
# 7. TWEAK TOBACCO — enhance your commercial tobacco
#    Your tobacco is too dry (not enough smoke)? Lacks flavor? Needs
#    menthol or extra sweetness? Add glycerin for thicker smoke, flavor
#    drops or aroma concentrates, menthol/cooling agents, honey, molasses,
#    water, or even nicotine. The calculator guides you step by step with
#    recommended amounts for each additive.
#
# 8. DIY TOBACCO — make shisha tobacco from scratch
#    Create your own tobacco or herbal blend from raw ingredients. Enter
#    the amount of tobacco (Virginia, Burley, or herbal), choose the
#    tobacco-to-molasses ratio (4:1 for dry, 3:1 standard, 2:1 moist,
#    1:1 very wet). Select sweetener (molasses, honey, or agave), set
#    glycerin amount, pick flavors from the database with auto-percentage.
#    Optional: add nicotine. Calculator computes the full molasses recipe
#    with exact ml and grams + recommended steeping time.
#
# 9. MOLASSES MIX — prepare liquid molasses base in bulk
#    Make a batch of liquid molasses that you later mix with tobacco.
#    Enter total amount, choose sweetener type, set glycerin proportion,
#    pick flavors, add water if needed. The calculator shows exact
#    quantities and tells you "Enough for X grams of tobacco at 3:1
#    ratio" so you know how far your batch will go.
#

# ============================================
# SAFETY & PREPARATION
# ============================================
# When mixing e-liquid, always wear protective gloves and work in a
# well-ventilated area, especially with nicotine. Use only quality
# ingredients with food-grade certification from verified suppliers.
#
# Ingredients you need for e-liquid:
# - Propylene Glycol (PG) — carries flavor, creates throat hit
# - Vegetable Glycerin (VG) — produces thick vapor, adds sweetness
# - Flavor concentrates — specifically designed for vaping (not food!)
# - Nicotine booster or nicotine salt (optional)
# - Medical gloves (nitrile or latex)
# - Empty glass bottle (dark amber for UV protection)
# - Measuring syringe or pipette for precise dosing
#
# Ingredients for hookah/shisha:
# - Vegetable Glycerin (VG) — food-grade, 99.5%+ purity
# - Molasses or honey — sweetener and binder
# - Tobacco leaves (Virginia, Burley) or herbal blend (nicotine-free)
# - Hookah bowl and heat management device (HMD)
#
