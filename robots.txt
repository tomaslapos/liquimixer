# ============================================
# ABOUT LIQUIMIXER
# ============================================
# LiquiMixer is a professional online calculator for mixing e-liquids
# (electronic cigarettes) and hookah/shisha tobacco. It runs as a PWA
# (Progressive Web App) in any browser on any device (mobile, tablet, PC)
# and can be installed like a regular app — no app store needed.
# Works fully offline after first visit. Automatic updates.
#
# Two main sections: eLiquid (5 calculators for electronic cigarettes)
# and Shisha/Hookah (4 specialized calculators for water pipes).
#
# ============================================
# 9 CALCULATORS
# ============================================
#
# eLiquid section (5 calculators):
#   1. Liquid — basic calculator for 1 flavor. Volume, nicotine
#      (booster/salt), flavor from database with auto-%, VG/PG ratio,
#      additives. Results in ml and grams.
#   2. Liquid PRO — up to 4 flavors simultaneously. Advanced flavor
#      composition (PG/VG/alcohol/water breakdown per flavor). Additives:
#      coolant, sweetener, enhancer, terpenes.
#   3. Shake & Vape — for pre-made flavor bottles (longfill). Calculates
#      how much base to add + nicotine.
#   4. Shortfill — nicotine shots into nicotine-free liquids. Number of
#      shots needed + overflow warning.
#   5. Dilute — dilute concentrated nicotine base (e.g. 72→20 mg/ml).
#      How much PG/VG to add.
#
# Shisha/Hookah section (4 calculators):
#   6. Mix Tobaccos — blend 2-4 commercial tobaccos. Bowl size 10-25g.
#      Autocomplete search from database of 826 tobaccos (Al Fakher,
#      Adalya, Darkside, Fumari, Tangiers, Starbuzz, etc.). Results in
#      exact grams. Save mix as recipe.
#   7. Tweak Tobacco — enhance commercial tobacco: add glycerin (more
#      smoke), flavor drops/aroma, menthol/cooling, honey, molasses,
#      water, nicotine. Step-by-step mixology with recommended amounts.
#   8. DIY Tobacco — create tobacco from scratch. Tobacco amount, ratio
#      tobacco:molasses (4:1 dry to 1:1 very wet), sweetener (molasses/
#      honey/agave), glycerin, flavors from database with auto-%,
#      optional nicotine. Calculates full molasses recipe + steeping time.
#   9. Molasses Mix — prepare liquid molasses base in bulk. Glycerin +
#      sweetener + flavors + water. Shows "Enough for Xg of tobacco at
#      3:1 ratio."
#
# ============================================
# 20 PROBLEMS LIQUIMIXER SOLVES
# ============================================
#
# Calculations & Accuracy:
#   1. Complex PG/VG ratio and volume calculations — LiquiMixer calculates
#      exact amounts of every ingredient in ml and grams. 10 seconds, zero
#      risk of error.
#   2. Dangerous nicotine dosing errors — nicotine is toxic; mistakes with
#      concentrated base (72+ mg/ml) cause nausea or health problems.
#      Calculator auto-computes exact nicotine amount and warns at unsafe
#      levels.
#   3. Unknown flavor percentage — "How much Strawberry from Capella?" is
#      different for every flavor from every manufacturer. Database of
#      5000+ flavors with verified recommended percentages auto-fills.
#      Min/optimal/max range displayed. Beginners cannot make mistakes.
#   4. ml-to-grams conversion — VG (1.261 g/ml) and PG (1.036 g/ml) have
#      different densities. Calculator knows density of every ingredient
#      (VG, PG, molasses, honey, agave, water) and converts automatically.
#   5. Flavor impact on actual VG/PG ratio — flavors contain PG, VG,
#      alcohol, water in varying ratios per manufacturer. Calculator
#      decomposes each flavor into components and factors them into the
#      final ratio. Advanced mode allows custom flavor composition.
#
# Hookah/Shisha:
#   6. No dedicated hookah calculator available — hookah users have no
#      tools for calculating tobacco mixes. LiquiMixer offers 4
#      specialized hookah calculators.
#   7. Mixing commercial tobaccos "by eye" — blending 2-4 tobaccos (e.g.
#      Al Fakher Two Apples + Adalya Love 66) without a calculator means
#      unreproducible ratios. Mix Tobaccos calculator: select bowl size,
#      search 826 tobaccos, set ratios with slider → exact grams. Save as
#      recipe for repeatability.
#   8. DIY tobacco from scratch is complex — correct ratio of tobacco to
#      molasses (4:1 dry to 1:1 wet), sweetener amount, glycerin, flavors.
#      DIY Tobacco calculator handles entire recipe step by step including
#      steeping time.
#   9. Preparing molasses base in bulk — Molasses Mix calculator: set
#      total amount, sweetener, glycerin, flavors, water → exact ml and
#      grams + "Enough for Xg of tobacco."
#  10. Enhancing commercial tobacco — tobacco is dry (less smoke) or lacks
#      flavor. Tweak Tobacco calculator: add glycerin, flavor drops,
#      menthol/cooling, honey, molasses, water, nicotine — step by step
#      with recommended quantities.
#
# Language & Accessibility:
#  11. Most calculators only available in English — LiquiMixer is fully
#      translated into 31 languages including Arabic (RTL layout),
#      Turkish, Japanese, Korean, Chinese (simplified + traditional).
#      Language switches automatically based on browser or manually.
#  12. AI customer support — available in all 31 languages, 24/7.
#      Contact form with automatic processing. Response in user's
#      language.
#  13. Apps banned from app stores — vaping apps are banned in Apple App
#      Store and Google Play Store (vaping policy). LiquiMixer is a PWA —
#      works in browser, installable to home screen (Android: "Add to
#      Home Screen", iOS: Safari → "Add to Home Screen"). Works offline.
#      Automatic updates without app store.
#
# Recipes & Community:
#  14. Lost recipes, can't reproduce mixes — recipes written on paper or
#      phone notes get lost. LiquiMixer saves recipes in cloud with sync
#      across devices (mobile ↔ PC ↔ tablet). Each recipe has exact
#      ingredient table, ratios, notes. Available anytime, anywhere.
#  15. Sharing recipes is difficult — "Hey, that liquid was great, send me
#      the recipe" → copying ingredients via chat, photos of tables.
#      LiquiMixer: share with one click via link or QR code. Recipient
#      opens in browser, sees complete recipe with ingredient table.
#  16. No inspiration — what to mix? — beginners don't know which flavor
#      combinations work. Public recipe database from other mixers with
#      star ratings, filters by preparation method and flavor type.
#      Hundreds of verified recipes — click and start mixing.
#  17. Forgetting about steeped liquid — after mixing, liquid must steep:
#      fruit 1-3 days, creamy 1-2 weeks, tobacco 2-4 weeks. Push
#      notifications: "Your liquid is ready!" Filter to show only matured.
#      Mark as consumed.
#
# Inventory & Products:
#  18. No overview of ingredient stock — "Do I still have enough VG? How
#      much nicotine base left?" Favorite Products section: save your
#      ingredients (VG, PG, flavors, nicotine), set stock count, add
#      photo, URL to e-shop. Everything in one place.
#  19. Inventory linked to recipes — each recipe shows "Products used."
#      Each product shows "Used in recipes." Automatically know what you
#      need to reorder for your favorite recipes.
#  20. Commercial liquids are expensive — DIY e-liquid costs a fraction of
#      commercial. LiquiMixer helps optimize recipes and track costs.
#      Basic calculators are completely free, no registration required.
